Решение задач по шифру Цезаря

1. Определение ключей шифра Цезаря

а. АПЕЛЬСИН → САЦЬНВЩЮ

Ключ: k = 17

Проверка: каждая буква сдвинута на 17 позиций в алфавите из 32 букв (без «Ё»).

б. АБРИКОС → ЫЬЛГЕЙМ

Ключ: k = 27

2. Расшифровка сообщений

а. ФХНЗКЧ

Расшифровка: ПРИВЕТ (ключ k = 5)

б. ЦЩЕБФ

Расшифровка: ВЕСНА (ключ k = 20)

3. Шифрование фамилии

Фамилия: Лосев

Ключ: k = 10

Зашифрованный результат: ХШЫПМ

4. Исходный код (JavaScript)

Для справки — логика шифрования и дешифрования:

// Алфавит без "Ё" — 32 буквы
const ALPHABET = 'АБВГДЕЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ';

function caesarEncrypt(text, k) {
    return text
        .toUpperCase()
        .split('')
        .map(char => {
            const idx = ALPHABET.indexOf(char);
            if (idx === -1) return char;
            return ALPHABET[(idx + k) % 32];
        })
        .join('');
}

function caesarDecrypt(text, k) {
    return text
        .toUpperCase()
        .split('')
        .map(char => {
            const idx = ALPHABET.indexOf(char);
            if (idx === -1) return char;
            return ALPHABET[(idx - k + 32) % 32];
        })
        .join('');
}

// Примеры:
console.log(caesarEncrypt('ЛОСЕВ', 10)); // "ХШЫПМ"
console.log(caesarDecrypt('ФХНЗКЧ', 5)); // "ПРИВЕТ"
console.log(caesarDecrypt('ЦЩЕБФ', 20)); // "ВЕСНА"
